# Dice Roller

A minimal, animated dice tool that rolls two six-sided dice and keeps a short history of results.

## Features
- 2D rolling animation with randomized faces during the roll.
- Recent roll history with totals and timestamps.
- Light/dark mode toggle that respects system preference by default.
- Single-file HTML with inline CSS/JS.

## Usage
1. Open `dice-roller.html` in a browser.
2. Click "Roll the dice" to animate and log a new result.
